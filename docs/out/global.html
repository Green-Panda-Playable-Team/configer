button(title, label, innerHTML, bootstrap, click, onClick, onChange)

Description:

Source:
  • ButtonBlock.js, line 10
Example

   type: "button",

   },

}
Properties:
Name Type Description
title string button label will be shown inside
label string same as title
innerHTML string field to specify custom button content (icon for example)
bootstrap string field to specify button class from bootstrap library
click function function that will be called when the user clicks on the button
onClick function the same as click
onChange function the same as click

data_set(title, description, value, min, max, step, snap, snap_threshold, onChange)

Description:
  • This complex type of option is intended for specifining of set of any input field for the user. Supports: text number / range color checkbox If you specify min or max property, then number type will be replaced with range type. If you specify value as "#f42344" - then this option will be built as a color field.
Source:
  • DataSetBlock.js, line 12
Example
TEXT_CONFIGURATION: {

   type: "data_set",

   title: "Logo text configuration",

   value: {

      number: 0, 

      string: "Hello", 

      boolean: true, 

      color: "#1abc9c"

   },

   onChange: function ( value ) {

      console.log( value )

   }

}
Properties:
Name Type Description
title string label that will be show next to the option
description string brief description of options, that will be shown when user hover over option
value object json object format of data set, from which will be built set of fields for the user. You can specify any json object.
min number minimum value that user can set for each number type.
max number maximum value that user can set for each number type.
step number step of increasing or decreasing number for each number type.
snap Array.<number> set of numbers to specify points to snap input cursor to these points for each range type.
snap_threshold number threshold of value to snap it to set point for each range type.
onChange function function that will be called when the user changes value of the option

list(title, description, list, template, onChange)

Description:
  • This is a special type of option indeed for specifying variable size of options set. You can specify the template parameter, and each value from the list will build with template options. After your list element will be ready, the list object will have new methods: add: function(value) { will append new item } remove: function(index) { will remove item by index } clear: function() { will clear your list } refresh: function() { fully refresh your list, when you set new list as LIST.list = [v1, v2] }
Source:
  • ListBlock.js, line 13
Example
LIST: {

    type: "list",

    title: "Configurations list",

    expanded: true,

    list: ["def_1", "data_2"],

    template: {

        type: "data",

        value: "",

        title: "Configuration {N}"

    },

    onChange: function(list) {

        console.log(list.length);

    }

    }
Properties:
Name Type Description
title string message that will be shown to the user as a running text.
description string brief description of options, that will be shown when user hover over option
list Array array object that contains all items. Each element can be either plain value, or object. You can specify different types of items.
template this is a template item that will be set as default for each item, if it is specified as a plain value.
onChange function function that will be called when the user changes value of the option
