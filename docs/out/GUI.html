GUI

configer GUI

new GUI(id, options)

Description:
  • A class for build GUI from config object.
Source:
  • GUI.js, line 25
Example
var config = {
	name: {
		type: "text",
		title: "Your name",
		value: "Bob",
	},
	
	age: {
		type: "number",
		title: "Your age",
		value: 22,
	},

	skill: {
		type: "select",
		title: "Choose your skill",
		options: ["js", "java", "c++"],
		labels: ["Javascript", "Java", "C++"],
		value: "js",
	}
};

configer.GUI("parent_element_id", {
	parent: "config-parent-element",
	config: config,
});
