datetime

Namespace

datetime

Description:
  • This type of option is intended for specifying the banner with running text. It is a decoration type, and can be used for a more beautiful setting of options.
Source:
  • DateBlock.js, line 4
Example
DATE: {

        type: "date",

        title: "Birthday:"

        value: "2022-12-10T18:20:04.138Z"
    }
Parameters:
Name Type Description
title string message that will be shown to the user as a running text.
description string brief description of options, that will be shown when user hover over option
value string default value date string of your option.
oninput boolean if set to true onChange callback will be triggered oninput otherwise onchange.
onChange function function that will be called when the user changes value of the option
